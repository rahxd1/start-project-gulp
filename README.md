# Gulp Starter for project

This repo is a gulp starter.

**Steps**
0-. Install Node for (npm)
1-.Install Gulp
  $ npm install gulp-cli -g
  $ npm install gulp -D
  --optional--
  $ touch gulpfile.js
  $ gulp --help

2-Install instaces before cloning
  $ npm install
  run
  $ gulp watch
  Use css with Sass folder scss
